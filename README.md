# Speed Ball

![Unity](https://img.shields.io/badge/Engine-Unity-black?style=flat&logo=unity)
![Language](https://img.shields.io/badge/Language-C%23-blue?style=flat&logo=csharp)

A high-speed arcade game developed in Unity. Control the ball, maintain your momentum, and navigate through obstacles in this test of reflexes.

**[Play the Game on Itch.io](https://volkanturkut.itch.io/speed-game)**

## ‚ö° Gameplay

**Speed Ball** focuses on velocity and precision.
* **Objective:** Navigate the course without losing momentum or crashing.
* **Mechanics:** Physics-based movement where speed is key.
* **Challenge:** The faster you go, the harder it is to control.


2.  Open **Unity Hub**.
3.  Add the project folder.
4.  Open the main scene in `Assets/Scenes`.
